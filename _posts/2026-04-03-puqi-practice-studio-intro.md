想学的东西列了一长串，真正记住、能用的却不多——很多时候我们停在「看过」「收藏过」，离「掌握」还差一截。**掌握**不是一次顿悟，**练习**恰恰是把它走通的过程：把知识**练进判断与反应**，一次次逼近「真的会」。**璞奇（Pracmo）**围绕「练」来设计：让 AI 陪你把材料拆成题目、陪你走完一轮练与复盘；写下来的笔记、做错的题有处可去；数据上云之后，换机换设备也不丢档。下面按产品主线，把这套「随身练习室」摊开来说一说。

## 开篇：为什么你需要「会练」的 App

知识很少是听一遍就内化的，**练一遍**才更像自己的。璞奇把三件事串成一条链：**流炼**里刻意练习、**笔记与错题**里沉淀与复盘、**云端同步**里让进度跟着人走。目标很简单——把学习从「看过」拉向「掌握」：练习是过程，掌握是方向；练什么、练多少、离「会了」还有多远，都能被看见。

![璞奇首页](https://blog.zendong.com.cn/assets/images/2026/2026-04-03-10-59-17.png)

## 一、流炼：刻意练习、快速上手，任何主题都能开练

**流炼**是璞奇的核心：把你想学的主题，变成一套**可反复练、可追踪**的练习流。

- **刻意练习，不是随机刷题**：围绕目标主题组织练习，练的是「会用的」而不是「看过的」。
- **路径短**：从「想练什么」到「开始答题」，步骤少，适合碎片时间开一局。
- **主题不限**：职业、考试、语言、兴趣……只要你想练，都可以和 AI 一起把内容拆成可练的题。
- **对话里完成学习与练习**：在流炼对话中，AI 帮你把材料变成题目、陪你走完练与复盘。

![多次交流生成流炼](https://blog.zendong.com.cn/assets/images/2026/2026-04-03-11-04-19.png)

![流炼练习详情](https://blog.zendong.com.cn/assets/images/2026/2026-04-03-11-05-22.png)

题型上支持**单选、多选、判断**等常见形式；**简答题**也完整支持——写完后由 **AI 判题**并给解析，让你知道哪里对、哪里要补。练习时可以边练边生成；题目会经 **AI 生成与审核**后再进入可用状态。

## 二、笔记与错题：学完有痕迹，错一次不白错

- **笔记**：支持笔记本与笔记管理，学习过程中的灵感、重点、复盘可以**沉淀成自己的知识库**，和练习形成闭环。

![笔记列表](https://blog.zendong.com.cn/assets/images/2026/2026-04-03-12-10-21.png)

- **错题**：支持分组、按题型查看与统计，方便**复盘回顾**——侧栏中的「复盘回顾」与错题本打通，让错题从「看过答案」推进到「下次做对」。

![错题回顾](https://blog.zendong.com.cn/assets/images/2026/2026-04-03-12-11-32.png)

## 四、通过 Agent Skill 接入「流炼」

如果你已经在用 **OpenClaw**、**Claude Code** 等 Agent 环境，可以把「当前对话里感兴趣的内容」整理成**流炼练习草案**，确认题型与题量后，由 Skill 调用开放平台创建练习，并返回分享链接；再到手机端打开，整条路径可以很顺。

**大致步骤：**

1. 在 [API Key 页面](https://www.zendong.com.cn/app/api-key) 登录并获取你的 API Key。
2. 在运行 Agent 的环境中设置环境变量 **`PRACMO_APIKEY`**（值为你的 Key；请勿把密钥贴进公开聊天）。

然后将如下内容贴给你的Agent，让它搞定安装：
```
参考如下命令 npx skills add https://github.com/zendong/skills --skill pracmo-flow-create 下载pracmo-flow-create skill内容到你的skill目录，然后读取此skill的SKILL.md，说明你新增的能力以及依赖
```

配置完成后，在 Agent 里一句话就能把当前材料收成流炼练习，并拿到可分享的链接：

![在 Claude Code 中生成流炼](https://blog.zendong.com.cn/assets/images/2026/2026-04-03-13-48-29.png)

链接可直接在浏览器里开练，也可以到手机端打开同一条流炼：
